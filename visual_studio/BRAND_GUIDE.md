# TripTalk Visual Brand Guide

## Core Identity
- Harm-reduction psychedelic education app
- Premium, Calm/Headspace tier design
- Warm, safe, professional — NEVER recreational or "trippy"

## Color Palette
- Primary gradient: deep forest green (#0A2F1F) → teal (#1A6B5A)
- Accent: warm champagne/gold (#D4AF37, #F5E6CC)
- Background: near-black (#0D1117)
- Text: warm white (#F5F5F0)

## Visual Rules
- NO text/words/labels in any generated image
- NO neon glow, NO bioluminescence effects
- NO psychedelic patterns, fractals, or kaleidoscope effects
- NO recreational imagery (smoking, consumption, parties)
- Mushroom strains: photorealistic DSLR macro only
- Botanicals (cacti, vines): photorealistic nature photography
- Clinical (ketamine): clean, warm, modern medical aesthetic
- Abstract/conceptual: soft, organic, minimal — think Headspace not Grateful Dead
- All images must work as hero banners (16:9) with text overlaid on top

## Ideogram API Notes
- Use `aspect_ratio: 16x9` for all hero/banner images
- Use `aspect_ratio: 1x1` for square assets (empty states)
- Use `rendering_speed: TURBO`
- Avoid terms that trigger safety filters (rephrase substance names if needed)
- Always include "no text, no words, no labels" in prompts
